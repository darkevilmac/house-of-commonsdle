# House of Commonsdle

A Vue 3 application for guessing members of the Canadian House of Commons.

## Attribution

Data provided by api.openparliament.ca.

## Tech Stack

- Vue 3
- Vite
- TailwindCSS
- Bun

## Project Setup

```sh
bun install
```

### Compile and Hot-Reload for Development

```sh
bun dev
```

### Type-Check, Compile and Minify for Production

```sh
bun run build
```

### Run Unit Tests with Vitest

```sh
bun test:unit
```

### Lint with ESLint

```sh
bun lint
```
